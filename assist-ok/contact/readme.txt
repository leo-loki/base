【設置方法】

このフォルダ内のファイルと「inc」フォルダ内の「setup.php（サイト情報セットファイル）」を利用します。

（１）このフォルダと「inc」フォルダは、サイトの最上階層へ設置してください。

（２）「inc」フォルダ「setup.php（サイト情報セット）」ファイルの内容をサイトの条件に合わせて修正してください。

（３）このフォルダ「form.php」ファイルの１４〜２１行目、
　　　/* FORM DETAIL SETUP */ 項目を適切なものに修正してください。

（４）「form.php」ファイルの必要ない入力項目（<tr>から</tr>まで）を削除します。
　　　「type="hidden"」のINPUTタグ項目は消さないでください。

（５）（４）で入力必須項目を削除した場合、
　　　「form.php」ファイルの３８〜４８行目の中から同じNAME属性の行を削除します。
　　　その際、最後の行末に「，」があれば消してください。

　　　（例）
　　　　radio: "radio",
　　　　texts: "required",
　　　　}
	　　　　　　　↓
　　　　radio: "radio",
　　　　texts: "required"
　　　　}

予め設置するサーバーの仕様をご確認ください。
